## Tại sao chọn Java?

Java là ngôn ngữ phổ biến cho backend và ứng dụng mạng.

### Ưu điểm
1. **Đa nền tảng** - Chạy trên mọi hệ điều hành
2. **Hiệu suất cao** với JVM
3. **Thư viện mạng phong phú**
